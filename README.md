# Mad-Finager
### A virtual file managment system can perform following: ###
##### 1. Create File #####
##### 2. Delete File #####
##### 3. Open File #####
##### 4. Read File #####
##### 5. Write File #####
##### 6. Close File #####

It represent its output and store its files in Javascript Object Notation(JSON) file in a very beautiful manner.
